# Role of AI in This Assignment

During the completion of this assignment, artificial intelligence (AI) was used as a support tool to better understand Java programming concepts. It helped provide clear explanations of basic notions such as classes, objects, constructors, as well as getters and setters methods.

AI also helped identify and correct errors in the code, including syntax issues, data type errors, and missing elements such as semicolons. Thanks to this, it was possible to gradually improve the program and make it functional.
